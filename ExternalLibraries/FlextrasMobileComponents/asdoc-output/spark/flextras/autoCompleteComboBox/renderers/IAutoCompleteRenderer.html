Packagespark.flextras.autoCompleteComboBox.renderers
Interfacepublic interface IAutoCompleteRenderer extends IDataRenderer, IItemRenderer, IVisualElement, ILayoutElement, ILayoutDirectionElement
Implementors DefaultAutoCompleteRenderer

This is an interface that defines the itemRenderer used in the Flextras Spark AutoCompleteComboBox It defines the typeAheadText, which can be used for creating highlighting in the itemRenderer and a reference to the AutoCompleteComboBox
